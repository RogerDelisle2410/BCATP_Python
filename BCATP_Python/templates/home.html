The Canadian Military Atlas is a comprehensive reference application documenting Canadian military bases, airfields, naval vessels, radar lines, historical installations, and related infrastructure. It provides a unified, searchable, and visually organized interface for exploring Canada’s military history — past and present.

Technology Stack

  • Backend: Python (Flask)
  • Frontend: HTML, CSS, JavaScript, Jinja templates
  • Database: SQLite
  • Deployment: Azure App Service
  • Version Control: Git + Azure DevOps

Features

  • Tabbed navigation for Army, Air Force, Navy, Ships, Dew Line, Pinetree, etc.
  • Global search with auto‑highlight and scroll‑to‑result
  • Image modal viewer for ships, planes, and tanks
  • Editable entries with clean modal forms
  • Pagination and alphabetical filtering
